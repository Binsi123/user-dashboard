# User Management Dashboard

A simple React application to display, search, filter, and manage users using data from a public API. Users can also be added locally via a modal form.

---

## Features

- Fetch users from public API
- Display users as cards
- Search users by name or username
- Filter users by company
- View user details on a separate route
- Add new users locally using a modal form
- Form validation using React Hook Form
- Responsive UI using Tailwind CSS

---

## Tech Stack

- React (Vite)
- React Router
- React Hook Form
- Tailwind CSS
- JavaScript (ES6+)

---

### Prerequisites
- Node.js (v18 or above)
- npm (comes with Node.js)
